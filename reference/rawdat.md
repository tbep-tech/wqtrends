# Raw data from San Francisco Estuary (South Bay)

Raw data from San Francisco Estuary (South Bay)

## Usage

``` r
rawdat
```

## Format

A `data.frame` object with 12411 rows and 8 columns

- date:

  Date

- station:

  int

- param:

  chr

- value:

  num

- doy:

  num

- cont_year:

  num

- yr:

  num

- mo:

  Ord.factor

## Details

Data from `datprc` object in <https://github.com/fawda123/SFbaytrends>
